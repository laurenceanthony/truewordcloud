# TrueWordCloud Examples

This folder contains example word cloud images generated by running `examples.py` from the root directory.

## Generating Examples

From the project root, run:
```bash
python examples.py
```

This will generate all 8 example images in this folder.

## Example Files

### Greedy Method (Spiral Layout)
- **example_greedy_basic.png** - Basic word frequencies (10 words)
- **example_greedy_colors.png** - Custom colors with red-blue gradient
- **example_greedy.png** - Comparison example showing greedy layout
- **example_greedy_custom_font.png** - Using a custom font (serif)

### Square Method (Compact Packing)
- **example_square_keyness.png** - Keyness scores (statistical significance)
- **example_square_categorical.png** - Categorical colors (languages, libraries, concepts)
- **example_square.png** - Comparison example showing square layout
- **example_square_large.png** - Large dataset with 50 words

## About the Methods

- **Greedy**: Fast, deterministic, creates radial/circular patterns
- **Square**: Compact, randomized, maintains roughly square aspect ratio
