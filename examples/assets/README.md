
# TrueWordCloud Example Outputs

This folder contains example word cloud images generated by running `examples.py` from the project root.

## How to Generate

Run:
```bash
python examples.py
```
This will generate a variety of example images in the `examples/` folder, demonstrating different features and layouts.

## About the Layout Methods

- **Greedy**: Fast, deterministic, spiral/radial pattern
- **Square**: Compact, randomized, fills gaps for a squarer aspect ratio
- **Distance Transform**: Most organic, best for irregular masks, denser packing

Compare the different layouts and features by viewing the generated images side by side.
